# chatgpt-api-playground

A command line tool for ChatGPT API.

## Requirements

- API Key

```
# https://help.openai.com/en/articles/4936850-where-do-i-find-my-secret-api-key
export OPENAI_API_KEY='YOUR PERSONAL API KEY'
```

- Python Package(s)

```
pip3 install openai
```

## Usage

```
python3 main.py
```
